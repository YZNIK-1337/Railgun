# Рельсотрон (Railgun)

Добавляет в игру кастомное оружие — рельсотрон с системой накопительных зарядов и автоперезарядкой.  
Маскируется под дробовик (`GunShotgun`) и полностью переопределяет механику стрельбы.

---

## Механика

- Держится и выглядит как `GunShotgun`, стандартная стрельба заблокирована
- Имеет **5 независимых зарядов**, каждый перезаряжается отдельно
- **ЛКМ (выстрел)** — выпустить заряд:
  - Тратит один доступный заряд
  - Выполняется `RaycastAll` по направлению взгляда на **50 метров**
  - **Первый живой игрок** на пути луча получает **50 урона** (тип `Shotgun`)
  - Визуальный эффект: яркий голубой `Primitive`-цилиндр вдоль луча, исчезает через `0.08 сек`
- **Автоперезарядка**: каждый потраченный заряд восстанавливается через **3 секунды** независимо от остальных
- HUD показывает состояние каждого из 5 зарядов:
  - `■` — заряд готов
  - `▨` — заряд перезаряжается (без таймера)
  - `3` / `2` / `1` — заряд перезаряжается, обратный отсчёт до готовности

---

## Установка

**Требуется [EXILED](https://github.com/ExMod-Team/EXILED)**

Поместите `RailgunPlugin.dll` в папку `EXILED/Plugins`

---

## Конфиг

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `is_enabled` | bool | `true` | Включить плагин |
| `debug` | bool | `false` | Выводить отладочные сообщения |
| `in_hand_hint` | string | `"В руке: ♂name♂\n♂desk♂\n♂extra♂"` | Формат хинта при держании |

### Параметры предмета (в коде класса `Railgun`)

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `CooldownDuration` | float | `3.0` | Время перезарядки одного заряда (секунды) |
| `RailgunDamage` | float | `50.0` | Урон по первой цели на пути |
| `ShotDist` | float | `50.0` | Максимальная дальность луча (метры) |
| `ClipSize` | byte | `5` | Количество зарядов |
